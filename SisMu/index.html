External CSS

#

Dengan external CSS, kamu hanya perlu satu file CSS untuk seluruh halaman di situsmu. Ini membuat pengaturan gaya lebih mudah dan konsisten di seluruh halaman.

Internal CSS

{}

Dengan internal CSS, kamu menaruh aturan gaya di bagian head dokumen HTML untuk mengatur tampilan halaman tersebut.

Inline CSS

</>

Inline CSS memungkinkan perubahan langsung ke satu elemen tertentu, seperti perubahan warna atau gaya tulisan.

Layout HTML

Header: Ini kepala dokumen atau bagian teratas. Seperti gerbang utama yang penuh wibawa.

Nav: Di sini terletak semua navigasi untuk para pengguna, seperti peta jalur menuju halaman lainnya.

Section: Digunakan untuk membagi isi dokumen menjadi beberapa bagian. Bayangkan seperti membagi ruangan besar menjadi beberapa ruang untuk berbagai keperluan.

Article: Bagian ini mandiri, seperti artikel di surat kabar. Dapat berdiri sendiri dengan maknanya, seperti pidato yang disampaikan kepada pengunjung.

Aside: Ini bagian tambahan, seperti catatan kaki atau sidebar yang memberikan informasi sekunder.

Footer: Bagian penutup di bagian bawah halaman, seperti kata terakhir saat acara selesai.

Details dan Summary: Elemen ini digunakan untuk menyembunyikan dan menampilkan detail, seperti membuka rahasia tersembunyi. Pengguna bisa klik untuk melihat lebih banyak informasi.
